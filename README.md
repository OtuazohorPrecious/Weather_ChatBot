# Weather API Microservice

A minimal FastAPI-based microservice that returns a simple weather forecast for a given city. Containerized using Docker.

## Features
- Single endpoint: `/weather?city=CityName`
- Hardcoded forecasts for demonstration
- Easily extendable to support external API integrations

## Requirements
- Python 3.9+
- Docker (if running in a container)
